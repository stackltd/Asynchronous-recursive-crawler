# Asynchronous-recursive-crawler
Асинхрнный краулер для рекурсивного поиска внутренних/внешних url

## Инструкция к асинхронному краулеру

### 1. Запуск краулера
Функция запуска:
async_crawler(url="ссылка", level=глубина, mode="режим")

Параметры:
* url: Стартовая страница (точка входа).
* level: Глубина погружения. 
    * 0: Только стартовая страница.
    * 1+: Краулер пойдет по найденным ссылкам дальше.
* mode: Фильтр ссылок.
    * "external": Ищет внешние ссылки
    * "internal": Ищет внутренние ссылки
    * "": Собирает все ссылки.

### 2. Результаты работы
* В консоли краткий отчет: сколько ссылок найдено, сколько ошибок таймаута возникло и сколько времени это заняло.
* В файле: В папке logs/ появится текстовый файл с названием ресурса, датой, временем и режимом поиска. Там будет список всех найденных ссылок, разбитый по уровням и страницам, на которых они были найдены.
* В репозитории в папке logs в качестве примера лежат результаты работы программы с некоторыми ресурсами
